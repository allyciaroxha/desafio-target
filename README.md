# Desafio Target

Repositório com os exercícios do processo seletivo.

## Arquivos

**comissoes_simples.py**  
Programa que calcula as comissões dos vendedores baseado nas vendas realizadas.

**estoque.py**  
Sistema para controle de movimentação de estoque (entrada/saída de produtos).

**juros.py**  
Calculadora de juros para contas em atraso (2,5% ao dia).

## Como executar

```
python nome_do_arquivo.py
```

Os dados estão nos arquivos JSON correspondentes.
